# Indian Knowledge System

A cinematic and immersive web experience exploring the depth of Indian civilisation through science, mathematics, astronomy, medicine, architecture, philosophy, naval engineering, music, and cultural achievements.

This project presents Indian knowledge systems through an interactive timeline-based storytelling interface with animated visuals, multilingual support, SVG artwork, particle systems, and a premium historical aesthetic.

---

# Overview

Indian Knowledge System is a single-page interactive historical experience designed to showcase major contributions of Indian civilisation across thousands of years.

The experience combines:

- Historical storytelling
- Animated transitions
- Interactive navigation
- Responsive design
- Sanskrit and Hindi integration
- Timeline-based exploration
- Custom SVG illustrations
- Ambient particle effects
- Cinematic intro animation

The design philosophy focuses on elegance, atmosphere, and immersion rather than traditional webpage layouts.
